# BeSBench Aggregated Benchmark Catalog

This table provides a detailed mapping of test cases aggregated within BeSBench from various source benchmarks. It includes classification according to the BeSBench taxonomy (Domain, Assessment Type) and links each test case to its origin.

**Columns:**

* `BenchmarkTool`: The source benchmark providing the test case.
* `Assessment Type`: Classifies the test as Intrinsic or Extrinsic.
* `Domain`: The primary risk domain (Safety, Security, Both, Other).
* `Category`: The main BeSBench risk category.
* `Subcategory`: The specific type of test or vulnerability.
* `Description`: Explanation of the test case's purpose.
* `Tags`: Keywords including source, domain, type, rationale (Reason:...), format, etc.
* `Dataset`: Link to or identifier for the underlying dataset/probe info, if available.
* `Playbook`: Associated BeS Playbook (Future Work).
* `Environment`: Associated BeS Environment (Future Work).
